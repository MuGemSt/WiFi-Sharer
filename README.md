## Usage
 1. Download and decompress the package of software
 2. Run _Sharer.exe_ as admin

## Build
 1. Clone the repo by Git
 2. Install Microsoft Visual Studio 2010 / 2013 / 2015
 3. Open _Sharer.sln_ as admin
 4. Perform Build action
